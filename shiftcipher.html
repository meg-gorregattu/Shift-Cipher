Shift Cipher

Enter a message and a shift value. Then, select whether you would like to encrypt or decrypt your message.

Encrypt: Takes your message and returns a capitalized message with each letter shifted forward in the alphabet based on the set shift value.

Decrypt: Takes your encrypted message and returns a lower case message with each letter shifted back in the alphabet based on the set shift value.

Shift Value: Shifts each letter forward in the alphabet by a given number. Use a number between 1 and 25.
